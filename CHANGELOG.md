# 更新日志

## 0.0.2

### ⭐ New Features | 新功能

1. AES 对称性加密（包含：仅秘钥、秘钥+偏移量、选择模式、选择填充方式）
2. RSA 非对称性加密（包含：生成指定位数的RSA密钥对，公钥加密，私钥加密，私钥解密）
3. IP（获取当前网络的 IP v4、IP v6）
4. TimeStamp（时间戳和时间互转）
5. IdCard（在线随机生成身份证号）
